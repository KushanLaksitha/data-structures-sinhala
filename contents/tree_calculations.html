Example 1: Binary Tree Height Calculation

Problem: Calculate the height of a binary tree

10 / \ 5 15 / \ / \ 3 7 12 20 / 1
Formula: Height = max(left_height, right_height) + 1
Base case: Empty node = -1, Leaf node = 0
Calculate from bottom up:
Node 1: Height = 0 (leaf)
Node 3: Height = max(0, -1) + 1 = 1
Node 7: Height = 0 (leaf)
Node 5: Height = max(1, 0) + 1 = 2
Node 12: Height = 0 (leaf)
Node 20: Height = 0 (leaf)
Node 15: Height = max(0, 0) + 1 = 1
Node 10 (root): Height = max(2, 1) + 1 = 3
‚úÖ Tree Height = 3 (longest path: 10 ‚Üí 5 ‚Üí 3 ‚Üí 1)

Example 2: Total Number of Nodes in Complete Binary Tree

Problem: A complete binary tree has height 4. How many nodes?

Formula: Total Nodes = 2^(h+1) - 1
Given: Height (h) = 4
Level breakdown:
Level 0 (root): 2^0 = 1 node
Level 1: 2^1 = 2 nodes
Level 2: 2^2 = 4 nodes
Level 3: 2^3 = 8 nodes
Level 4: 2^4 = 16 nodes
Total: 1 + 2 + 4 + 8 + 16 = 31 nodes
Or using formula:
Total = 2^(4+1) - 1
Total = 2^5 - 1
Total = 32 - 1 = 31 nodes
‚úÖ Total Nodes = 31

Example 3: Binary Search Tree (BST) Search Operation

Problem: Find value 17 in BST. Count comparisons.

20 / \ 10 30 / \ / \ 5 15 25 35 / \ 12 17
Search for: 17
Step 1: Compare 17 with 20 (root)
17 < 20 ‚Üí Go LEFT | Comparisons: 1
Step 2: Compare 17 with 10
17 > 10 ‚Üí Go RIGHT | Comparisons: 2
Step 3: Compare 17 with 15
17 > 15 ‚Üí Go RIGHT | Comparisons: 3
Step 4: Compare 17 with 17
17 == 17 ‚Üí FOUND! | Comparisons: 4
Path: 20 ‚Üí 10 ‚Üí 15 ‚Üí 17
‚úÖ Found at depth 3, Total comparisons = 4

Example 4: Leaf Node Count Calculation

Problem: Count leaf nodes in the tree

A / | \ B C D /| | |\ E F G H I
Definition: Leaf = Node with NO children
Check each node:
Node A: Has children B, C, D ‚Üí NOT a leaf
Node B: Has children E, F ‚Üí NOT a leaf
Node C: Has child G ‚Üí NOT a leaf
Node D: Has children H, I ‚Üí NOT a leaf
Node E: No children ‚Üí LEAF ‚úì (Count: 1)
Node F: No children ‚Üí LEAF ‚úì (Count: 2)
Node G: No children ‚Üí LEAF ‚úì (Count: 3)
Node H: No children ‚Üí LEAF ‚úì (Count: 4)
Node I: No children ‚Üí LEAF ‚úì (Count: 5)
‚úÖ Total Leaf Nodes = 5 (E, F, G, H, I)

Example 5: Binary Tree Sum of All Nodes

Problem: Calculate sum of all node values

50 / \ 30 70 / \ / \ 20 40 60 80
Traverse and sum all nodes:
Level 0: 50
Level 1: 30 + 70 = 100
Level 2: 20 + 40 + 60 + 80 = 200
Total Sum:
Sum = 50 + 100 + 200
Sum = 350
Alternative (node by node):
50 + 30 + 70 + 20 + 40 + 60 + 80 = 350
‚úÖ Sum of All Nodes = 350

Example 6: Maximum Width of Binary Tree

Problem: Find the maximum width (maximum nodes at any level)

1 / \ 2 3 / \ / \ 4 5 6 7 / 8
Count nodes at each level:
Level 0: [1] ‚Üí Width = 1
Level 1: [2, 3] ‚Üí Width = 2
Level 2: [4, 5, 6, 7] ‚Üí Width = 4
Level 3: [8] ‚Üí Width = 1
Compare all widths:
Level widths: [1, 2, 4, 1]
Maximum = max(1, 2, 4, 1) = 4
‚úÖ Maximum Width = 4 (at Level 2)

Example 7: Binary Tree Diameter Calculation

Problem: Find diameter (longest path between any two nodes)

1 / \ 2 3 / \ 4 5 / \ 6 7
Definition: Diameter = Longest path between any 2 nodes
Formula: diameter = left_height + right_height + 2
Check each node as potential center:
At Node 1:
Left height (2‚Üí4‚Üí6) = 2
Right height (3) = 0
Diameter through 1 = 2 + 0 + 2 = 4
At Node 2:
Left height (4‚Üí6) = 1
Right height (5) = 0
Diameter through 2 = 1 + 0 + 2 = 3
At Node 4:
Left height (6) = 0, Right height (7) = 0
Diameter through 4 = 0 + 0 + 2 = 2
Longest path: 6 ‚Üí 4 ‚Üí 2 ‚Üí 5 (or 7 ‚Üí 4 ‚Üí 2 ‚Üí 5)
Path length = 4 edges = 5 nodes
‚úÖ Diameter = 4 edges (5 nodes)

Example 8: AVL Tree Balance Factor Calculation

Problem: Calculate balance factor for each node

10 / \ 5 15 / \ 2 20
Formula: Balance Factor (BF) = Height(Left) - Height(Right)
Tree is balanced if: -1 ‚â§ BF ‚â§ 1 for all nodes
Calculate heights first:
Node 2: Height = 0 (leaf)
Node 5: Height = 1 (has one child)
Node 20: Height = 0 (leaf)
Node 15: Height = 1 (has one child)
Node 10: Height = 2 (root)
Calculate Balance Factors:
BF(2) = 0 - 0 = 0 ‚úì
BF(5) = 1 - 0 = 1 ‚úì
BF(20) = 0 - 0 = 0 ‚úì
BF(15) = 0 - 1 = -1 ‚úì
BF(10) = 2 - 2 = 0 ‚úì
‚úÖ Tree is BALANCED (all BF values: -1, 0, or 1)

Example 9: Binary Tree Level Order Sum

Problem: Calculate sum at each level and find maximum

10 / \ 20 30 / \ / \ 40 50 60 5
Sum nodes at each level:
Level 0:
Nodes: [10]
Sum = 10
Level 1:
Nodes: [20, 30]
Sum = 20 + 30 = 50
Level 2:
Nodes: [40, 50, 60, 5]
Sum = 40 + 50 + 60 + 5 = 155
Compare level sums:
Level sums: [10, 50, 155]
Maximum = 155 (at Level 2)
‚úÖ Maximum Level Sum = 155 (Level 2)

Example 10: Binary Tree Path Sum Calculation

Problem: Find all root-to-leaf paths and their sums

5 / \ 4 8 / / \ 11 13 4 / \ \ 7 2 1
Find all root-to-leaf paths:
Path 1: 5 ‚Üí 4 ‚Üí 11 ‚Üí 7
Sum = 5 + 4 + 11 + 7 = 27
Path 2: 5 ‚Üí 4 ‚Üí 11 ‚Üí 2
Sum = 5 + 4 + 11 + 2 = 22
Path 3: 5 ‚Üí 8 ‚Üí 13
Sum = 5 + 8 + 13 = 26
Path 4: 5 ‚Üí 8 ‚Üí 4 ‚Üí 1
Sum = 5 + 8 + 4 + 1 = 18
Statistics:
Total paths = 4
Maximum sum = 27 (Path 1)
Minimum sum = 18 (Path 4)
Average sum = (27 + 22 + 26 + 18) / 4 = 93 / 4 = 23.25
‚úÖ Path Sums: [27, 22, 26, 18] | Max: 27, Min: 18, Avg: 23.25
