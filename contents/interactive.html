Example 1: Expression Evaluation using Stack

Problem: 5 + ((1 + 2) * 4) - 3 = ?

Step 1: Parse expression left to right
Step 2: Inner brackets: (1 + 2) = 3
Stack: [5] ‚Üí Push 5
Step 3: Calculate: 3 * 4 = 12
Stack: [5, 12]
Step 4: 5 + 12 = 17
Stack: [17]
Step 5: 17 - 3 = 14
Result: 14

Example 3: CPU Task Scheduling using Queue

Scenario: 4 Tasks waiting for CPU

Initial: Tasks = [Task1, Task2, Task3, Task4]
Queue: [T1] ‚Üê [T2] ‚Üê [T3] ‚Üê [T4]
CPU executes: Dequeue() ‚Üí T1 (2ms)
Queue: [T2] ‚Üê [T3] ‚Üê [T4]
CPU executes: Dequeue() ‚Üí T2 (3ms)
Queue: [T3] ‚Üê [T4]
New task arrives: Enqueue(T5)
Queue: [T3] ‚Üê [T4] ‚Üê [T5]
Total time: 2ms + 3ms = 5ms (T1 + T2)

Example 6: Balanced Parentheses Check using Stack

Problem: Check if "{[()]}" is balanced

Input: { [ ( ) ] }
Process each character:
Read '{' ‚Üí Opening bracket, Push('{') ‚Üí Stack: ['{']

Read '(' ‚Üí Opening bracket, Push('(') ‚Üí Stack: ['{', '[', '(']

Stack: ['{', '[']
Read ']' ‚Üí Closing bracket, Pop() = '[' ‚úì Match!
Stack: ['{']
Read '}' ‚Üí Closing bracket, Pop() = '{' ‚úì Match!
Stack: [] (Empty)
Result: ‚úÖ BALANCED (Stack is empty)

Example 8: Function Call Stack Memory

Problem: Calculate factorial(4) recursively

Call: factorial(4)
Push(factorial(4)) ‚Üí Stack: [f(4)]
f(4) calls f(3) ‚Üí Push(factorial(3)) ‚Üí Stack: [f(4), f(3)]
f(3) calls f(2) ‚Üí Push(factorial(2)) ‚Üí Stack: [f(4), f(3), f(2)]
f(2) calls f(1) ‚Üí Push(factorial(1)) ‚Üí Stack: [f(4), f(3), f(2), f(1)]
f(1) = 1 (base case) ‚Üí Pop() ‚Üí Stack: [f(4), f(3), f(2)]
f(2) = 2 √ó 1 = 2 ‚Üí Pop() ‚Üí Stack: [f(4), f(3)]
f(3) = 3 √ó 2 = 6 ‚Üí Pop() ‚Üí Stack: [f(4)]
f(4) = 4 √ó 6 = 24 ‚Üí Pop() ‚Üí Stack: []
Result: factorial(4) = 24
Max Stack Depth: 4 function calls

Example 9: Round Robin CPU Scheduling

Scenario: 4 processes with time quantum = 3ms

Processes: P1(9ms), P2(5ms), P3(7ms), P4(4ms)
Time Quantum: 3ms per process
Initial Queue: [P1(9)] ‚Üê [P2(5)] ‚Üê [P3(7)] ‚Üê [P4(4)]
Round 1:
Dequeue P1, execute 3ms ‚Üí Remaining: 6ms ‚Üí Enqueue P1(6)
Queue: [P2(5)] ‚Üê [P3(7)] ‚Üê [P4(4)] ‚Üê [P1(6)]
Dequeue P2, execute 3ms ‚Üí Remaining: 2ms ‚Üí Enqueue P2(2)
Queue: [P3(7)] ‚Üê [P4(4)] ‚Üê [P1(6)] ‚Üê [P2(2)]
Round 2:
Dequeue P3, execute 3ms ‚Üí Remaining: 4ms ‚Üí Enqueue P3(4)
Dequeue P4, execute 3ms ‚Üí Remaining: 1ms ‚Üí Enqueue P4(1)
Continue until all complete...
Total time: 9 + 5 + 7 + 4 = 25ms
